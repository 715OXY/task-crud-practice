# task-crud-practice

## 概要

COACHTECH 教材 Tutorial 11-2「タスク管理APIのCRUD実装」で作成した成果物です。

タスク管理APIを実装し、Postmanで各操作のステータスコードとレスポンスの確認を行っています。

## 使用技術

- PHP 8.x
- Laravel 10.x
- REST API（CRUD）
- API Resources、FormRequest（バリデーション）

## 学んだこと

- データベースを使ったRESTful APIとCRUD操作の実装方法
- 適切なHTTPステータスコードの選択方法
- API Resourcesを使ってレスポンスを整形する方法
- Postman のインストール方法と設定利用方法
